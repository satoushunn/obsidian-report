# 夫婦資産管理戦略：最終報告書

**対象者**：26歳既婚ソフトウェアエンジニア夫婦（結婚日：2025年6月2日）  
**年収帯**：双所得世帯（5M JPY以上）  
**現在地**：仙台市 泉区（泉中央）  
**居住形態**：賃貸で二人暮らし  
**給与振込先**：夫妻両方とも七十七銀行  

---

## Executive Summary

夫婦資産管理において、以下の3つの層に資金を分離し、各層で最適なツールを組み合わせることが最適です。

| 層 | 資金性質 | 運用期間 | 管理方法 | 特性 |
| --- | --- | --- | --- | --- |
| **第1層** | 生活費 | 1ヶ月～数ヶ月 | 七十七銀行普通預金（共有口座） | 流動性・透明性重視 |
| **第2層** | 教育資金・車購入 | 5～15年 | 新NISA つみたて投資枠 | 安定成長・非課税 |
| **第3層** | 住宅資金・老後 | 10～30年 | 新NISA 成長投資枠 + iDeCo | 高成長・非課税 |

## 第1層：生活費管理（月26万円）

### 最適解：七十七銀行「家族口座見守りサービス + 代理人カード」

**理由**：
- 給与振込先をそのまま利用（新規口座開設不要）
- 七十七銀行の公式サービス（法的安全性が最高）
- 完全無料（全機能無料）
- 夫婦双方が平等にアクセス可能
- 手数料ゼロ運用が実現可能

### 実装方法

#### Step 1：夫名義の七十七銀行普通預金を「共有口座」として指定
```text
口座名義：夫
用途：生活費共同管理
通常利用：現状維持（給与振込口座として機能）
```

#### Step 2：妻に代理人カードを発行申請（窓口）

| 項目 | 詳細 |
| --- | --- |
| **来店場所** | 七十七銀行最寄り支店 |
| **来店者** | 夫と妻（両方必須） |
| **所要時間** | 15-30分 |
| **手数料** | 無料 |
| **持参物** | 夫の通帳・届出印・本人確認書類（両人分） |
| **カード到着** | 即日受取 or 郵送（数日） |

#### Step 3：家族口座見守りサービスをオンラインで設定

1. 夫が七十七銀行インターネットバンキングにログイン
2. 「お取引」 → 「家族口座見守りサービス」を選択
3. 妻の情報を登録（メールアドレス、続柄：配偶者）
4. 見守り範囲を設定：
   - 口座残高：「表示あり」
   - 入出金明細：「全取引を表示」
   - 通知：「全取引を通知」
5. 申込完了

所要時間：5-10分、手数料：無料

### 月額運用フロー

```text
【27日：給与振込（自動）】
夫給与 30万円 → 夫の七十七銀行普通預金
妻給与 20万円 → 妻の七十七銀行普通預金

【28日：資金集約】
妻が楽天銀行 → 夫の七十七銀行へ13万円を自動振込
（給与受取で月3回無料枠を利用 → 実質無料）

【日常運用（月初～月末）】
妻の日々の行動：
  ・毎日：スマホアプリで「家族口座見守りサービス」を確認
  ・確認内容：夫の口座残高、全ての入出金明細
  ・通知：メールで取引を即座に通知（全取引通知設定）

支出決済：
  ・カード決済：夫のデビットカード or 妻の代理人カード
  ・ATM出金：妻の代理人カードで対応
  ・予定外の入金：妻が楽天銀行から振込

【月末：透明性確認】
夫妻でアプリを見せ合い、その月の支出分析を共有
```

### 手数料試算（月26万円の生活費管理）

| 項目 | コスト | 根拠 |
| --- | --- | --- |
| 七十七銀行内振込（七十七銀行あて） | 0円 | 常に無料 |
| 妻の楽天銀行 → 夫の七十七銀行振込 | 0円 | 給与受取で月3回無料 |
| 代理人カード発行・維持 | 0円 | 無料 |
| 家族口座見守りサービス | 0円 | 完全無料 |
| 七十七銀行ATM利用（複数回） | 0円 | 通常無料 |
| **月間合計** | **0円** | **実質無料運用** |

### 重要な制限事項

1. **家族口座見守りサービスの対象は普通預金のみ**
   - 目的別預金は見守りサービスの対象外
   - 普通預金のみで運用することを推奨

2. **見守り情報の更新タイミング**
   - 見守り情報は1日6回の定時更新
   - 厳密にはリアルタイムではないが、メール通知で補完
   - 実務的にはほぼリアルタイムと同等

3. **七十七銀行への事前確認が必須**
   ```text
   確認内容：
   「代理人カード保有者がオンラインバンキングで
    直接出金操作できるか」
   ```
   金融機関の仕様により異なる可能性があるため、
   最寄り支店に事前確認が必須

## 第2層：教育資金・車購入資金（5～15年運用）

### 最適解：新NISA「つみたて投資枠」を活用

**対象資金**：
- 教育資金：500万円～1,000万円
- 車購入資金：200万円～500万円

**運用構成**：
- 妻：年120万円（つみたて投資枠で推奨）
- 夫：年120万円（つみたて投資枠で推奨）
- **夫婦合計：年240万円の非課税投資**

### 期待成果シミュレーション

**条件**：妻が月10万円を年3.5%で18年運用

| 項目 | 金額 |
| --- | --- |
| 投資元本（8万円×12月×18年） | 1,728万円 |
| 運用益（年3.5%複利） | 約570万円 |
| **合計** | **約2,298万円** |

このうちから：
- 教育資金 500万円を引き出し（2042-2045年の4年間で必要額取出）
- 運用を継続（売却枠が翌年復活）
- 残余資金で住宅資金、老後資金に充当

### 実装方法

**Step 1：夫婦それぞれが新NISA口座を開設（オンライン完結）**

| 項目 | 内容 |
| --- | --- |
| 証券会社 | SBI証券 または 楽天証券（推奨：手数料ゼロ） |
| 開設時間 | 各自で15-30分（オンライン完結） |
| 必要書類 | 本人確認書類（スマホ撮影でOK） |
| 手数料 | 無料 |
| 所要日数 | 最短3営業日 |

**Step 2：つみたて投資枠で毎月自動買付を設定**

| 項目 | 内容 |
| --- | --- |
| 投資対象 | インデックスファンド（S&P500またはオールカントリー） |
| 月額 | 夫妻各10万円（年120万円） |
| 買付方法 | 毎月初日の自動定額買付 |
| 非課税期間 | 生涯（上限1,800万円/人） |

**Step 3：銀行口座から証券口座への自動振替を設定**

```text
妻の楽天銀行 → SBI証券 / 楽天証券へ月10万円自動入金
夫のPayPay銀行 → SBI証券 / 楽天証券へ月10万円自動入金

※ 各証券会社と銀行の連携で完全自動化可能
```

### 投資対象（初心者向け推奨）

| 商品名 | リスク | リターン（目安） | 推奨度 |
| --- | --- | --- | --- |
| **S&P 500 インデックスファンド** | 中程度 | 年5～7% | ⭐⭐⭐⭐⭐ |
| オールカントリー指数ファンド | 中程度 | 年4～6% | ⭐⭐⭐⭐ |

### 注意事項

1. **市場下落時の心理的対応**
   - つみたて投資は「積立を続けることが最重要」
   - 市場が下落した月も淡々と積立続行
   - 20年の長期で見れば、下落は「買い増しの機会」

2. **売却タイミング**
   - 教育資金が必要になる18年後以降に段階的に引き出し
   - 生涯非課税枠（1,800万円/人）は温存推奨

3. **追加拠出の可能性**
   - ボーナス時期に追加投資（年240万円枠に余裕があれば）
   - 成長投資枠（年240万円）との併用で年480万円も可能

## 第3層：住宅資金・老後資金（10～30年運用）

### 最適解：新NISA「成長投資枠」+ iDeCo の組み合わせ

**対象資金**：
- 住宅資金（頭金）：1,000万円～2,000万円
- 老後資金：3,000万円以上

**運用構成**：
- 新NISA 成長投資枠：年240万円×2人 = 年480万円
- iDeCo（個人型確定拠出年金）：月3万円～5万円/人
- **合計年間投資：約600万円～720万円**

### 期待成果シミュレーション

**条件**：月20万円を年5%で30年運用

| 項目 | 金額 |
| --- | --- |
| 投資元本（20万円×12月×30年） | 7,200万円 |
| 運用益（年5%複利） | 約7,500万円 |
| **合計** | **約14,700万円** |

この資金から：
- 住宅購入時（10-20年後）に2,000万円引き出し
- 残余をさらに運用継続
- 60歳時点で老後資金として活用

### 新NISA + iDeCo の役割分担

| 項目 | 新NISA 成長投資枠 | iDeCo |
| --- | --- | --- |
| **年間上限** | 240万円/人 | 276万円/人（会社員向け） |
| **非課税期間** | 生涯 | 60歳まで（引き出し不可） |
| **投資対象** | 個別株、成長株、ETF等 | インデックスファンド、バランスファンド |
| **引き出しタイミング** | 自由（いつでも） | 60歳以降 |
| **税制優遇** | 利益非課税 | 所得控除 + 利益非課税 |

### 実装方法

**Step 1：新NISA成長投資枠での運用開始**

```text
夫：年240万円を成長投資枠で自由に投資
  対象：個別成長株、高配当株、グロース系ETF

妻：年240万円を成長投資枠で自由に投資
  対象：同上、またはバランス型

月額では：夫妻合計20万円/月の投資が可能
```

**Step 2：iDeCoの同時進行**

```text
夫：月5万円をiDeCoで積立
  運用対象：バランスファンド（リスク低～中）

妻：月3万円をiDeCoで積立
  運用対象：インデックスファンド

効果：
  ・給与所得から差し引かれ、所得税・住民税を軽減
  ・60歳以降に受け取る際も分割受取で税優遇
```

### 住宅購入時の引き出し戦略

**想定シナリオ**：35歳で住宅購入（10年後）

```text
【10年後のポートフォリオ想定】
新NISA 成長投資枠：2,400万円投資 → 推定3,100万円
iDeCo：960万円投資 → 推定1,200万円（ただし60歳まで不可引き出し）
その他貯蓄：2,000万円

【住宅購入資金の調達】

合計2,000万円で住宅頭金に充当
```

## 投資初心者向けの3つの心構え

### 1. つみたては「心理的負荷を減らすため」

毎月自動で投資信託を買い付けることで：
- 市場タイミングの判断をしない（心理的負荷軽減）
- 相場が下がった月も「より安く買える」と考える（ドルコスト平均法の効果）
- 20-30年でほぼ必ず利益が出る（歴史的実績）

### 2. 下落時は「引き出さない」が鉄則

市場が下落した時点での引き出しは損失を確定させます：
```text
❌ 相場下落 → 損切（引き出し）
✅ 相場下落 → 引き続き投資継続 → 数年後に回復で利益化
```

### 3. 分散投資で「卵を1つのカゴに入れない」

```text
第2層（つみたて）：　→ 低リスク（インデックスファンド）
第3層（成長投資）：　→ 中～高リスク（個別株・高配当株）
貯蓄・現金：　　　　→ ゼロリスク（いつでも引き出し可能）
```

## 法的・税務的な重要ポイント

### 1. 七十七銀行共有口座は「夫名義」で法的に完全安全

**相続時**：
- 夫が死亡した場合、妻が法定相続人として相続手続きで正規に引き継ぎ可能
- 遺産分割協議がスムーズ

**離婚時**：
- 婚姻中の共有財産として、50対50で分与対象
- 明確な根拠に基づいて分与額を計算可能

**破産時**：
- 預金保険機構の対象（最高1,000万円まで保護）

### 2. 新NISA は「夫婦で分ける」が原則

❌ **1つのNISA口座に夫婦で積立したい**
→ できません。NISA は個人単位の制度

✅ **夫と妻で別々のNISA口座を開設する**
→ 正解。非課税枠が年720万円（2人合計）になる

**贈与税への注意**：
- 夫婦間での資金移動（例：妻がPayPay銀行から夫の投資信託口座へ振込）
- 婚姻中は贈与税の対象外（相互扶助義務）
- ただし、婚外のパートナーや内縁の妻は贈与税対象

### 3. iDeCo は「節税効果が大きい」

加入時点での節税：
```text
給与 50万円（月額）
iDeCo 拠出：5万円
───────────────
課税所得：45万円（所得税・住民税が5万円分軽減）

年間：60万円 × 15% = 9万円の節税
```

## 付録：よくある質問への回答

### Q1：目的別預金も妻が見える化できる？
**A**：いいえ。家族口座見守りサービスは普通預金のみが対象です。目的別預金を活用する場合は、別途、夫婦で共有する家計簿アプリ（OsidOri等）で管理することを推奨。

### Q2：新NISAの枠を全部使い切らなかったら？
**A**：翌年に繰り越されます。ただし、枠の余裕があるので、ボーナスがあれば追加投資することも可能。

### Q3：市場が大きく下落したらどうする？
**A**：「買い増しのチャンス」と考えてください。安い時期にたくさん買える。20年の長期では、下落は必ず回復します。

### Q4：子どもが生まれたら？
**A**：2027年から「こども支援NISA」が始まります。子ども1人につき年60万円、生涯600万円の非課税投資ができます。教育資金準備に最適です。

### Q5：離婚した場合はどうなる？
**A**：七十七銀行の共有口座は夫名義のため、財産分与対象になります。新NISAは各自の個別口座のため、各自が保有を続けられます。ただし、婚姻中の利益部分は共有財産として分与対象になる可能性があります（弁護士に相談推奨）。
